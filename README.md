# ML Challenge Competition Solution

## 🎯 Overview
Complete machine learning solution for the IEEE ML Challenge competition with automated preprocessing, model training, and prediction generation.

## 📊 Dataset
- Training Data: `TRAIN.csv`
- Test Data: `test.csv`
- Output: `FINAL.csv`

## 🛠️ Technology Stack
- **Python 3.8+**
- **Scikit-learn** - ML algorithms
- **XGBoost** - Gradient boosting
- **LightGBM** - Fast gradient boosting
- **Pandas & NumPy** - Data manipulation
- **SMOTE** - Handling imbalanced data
